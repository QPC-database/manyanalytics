manyanalytics

This tool allows you to run a Google Analytics (GA) query across all websites in your GA account. You can build the query parameters using Google's own query tool then paste the parameters into this form to run the same query across multiple websites.

Your results can be saved to a CSV file for use in Excel or other applications.
